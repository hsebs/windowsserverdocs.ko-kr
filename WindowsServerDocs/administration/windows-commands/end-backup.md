# <a name="end-backup"></a>최종 백업




전체 백업 세션을 종료 시킵니다 문제는 **BackupComplete** 필요한 경우 적절 한 기록기 상태와 이벤트입니다.

## <a name="syntax"></a>구문

```
end backup
```

## <a name="remarks"></a>주의

-   이 명령은 모든 전체 백업에 대해 사용 해야 합니다.
