# <a name="sysocmgr"></a>sysocmgr

Sysocmgr, 않으며 Windows의 이후 릴리스에서 지원 되지 않을 수도 있습니다.

이 도구는 Windows Server 2003에 포함 됩니다. 자세한 내용은 참조 [Sysocmgr](https://technet.microsoft.com/library/cc773290(v=ws.10).aspx)합니다.
