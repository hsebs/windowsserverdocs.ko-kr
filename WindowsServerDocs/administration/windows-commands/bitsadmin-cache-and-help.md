# <a name="bitsadmin-cache-and-help"></a>bitsadmin cache and help

**캐시** 스위치에 대 한 명령줄 사용법을 표시 합니다.

## <a name="syntax"></a>구문

```
bitsadmin /cache /help
```

## <a name="examples"></a>예

**캐시** 스위치에 대 한 명령줄 도움말을 표시 합니다.

```
bitsadmin /cache /help
```
