# <a name="bitsadmin-cache-and-getlimit"></a>bitsadmin 캐시 및 getlimit

캐시 한계를 검색합니다.

## <a name="syntax"></a>구문

```
bitsadmin /cache /getlimit
```

## <a name="examples"></a><a name=BKMK_examples></a>예와

다음 예제에서는 캐시 제한 검색...

```
C:\>bitsadmin /cache /getlimit
```
