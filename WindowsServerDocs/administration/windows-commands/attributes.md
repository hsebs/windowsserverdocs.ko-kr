# <a name="attributes"></a>특성

표시 설정 하거나 디스크 또는 볼륨의 특성을 지웁니다.

## <a name="syntax"></a>구문

```
attributes disk
attributes volume
```

### <a name="parameters"></a>매개 변수

| 매개 변수 | 설명 |
| --------- | ----------- |
| [특성 디스크](attributes-disk.md) | 표시 설정 하거나 디스크의 특성을 지웁니다. |
| [특성 볼륨](attributes-volume.md) | 표시 설정 하거나 볼륨의 특성을 지웁니다. |
