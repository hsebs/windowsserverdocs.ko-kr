# <a name="begin-backup"></a>백업 시작

> 적용 대상: Windows Server 2012 R2, Windows Server 2012

전체 백업 세션을 시작 합니다. 이 명령은 기본 복사 백업 설정을 무시합니다.

## <a name="syntax"></a>구문

```
begin backup
```
