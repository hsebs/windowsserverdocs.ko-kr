# <a name="create"></a>create

하나 이상의 디스크 또는 가상 하드 디스크 (VHD)에 있는 볼륨으로 디스크에 파티션을 만듭니다.

## <a name="syntax"></a>구문

```
create partition
create volume
create vdisk
```
