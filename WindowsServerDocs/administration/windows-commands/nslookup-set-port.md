# <a name="nslookup-set-port"></a>nslookup set port



지정 된 값으로 기본 TCP/UDP 도메인 이름 시스템 (DNS) 이름 서버 포트를 변경 합니다.

## <a name="syntax"></a>구문

```
set port=<Port>
```

### <a name="parameters"></a>매개 변수

| 매개 변수 |                                          설명                                          |
|-----------|-----------------------------------------------------------------------------------------------|
|  \<포트>  | 기본 TCP/UDP DNS 이름 서버 포트에 대 한 새 값을 지정합니다. 기본 포트는 53입니다. |
|   {도움말   |                                              ?}                                               |
