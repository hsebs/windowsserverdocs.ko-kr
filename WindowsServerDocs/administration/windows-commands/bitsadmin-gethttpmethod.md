# <a name="bitsadmin-gethttpmethod"></a>bitsadmin gethttpmethod

작업에 사용할 HTTP 동사를 가져옵니다.

## <a name="syntax"></a>구문

```
bitsadmin /gethttpmethod <Job>
```

### <a name="parameters"></a>매개 변수

| 매개 변수 | 설명 |
| -------------- | -------------- |
| 제출 | 작업의 표시 이름 또는 GUID입니다. |
