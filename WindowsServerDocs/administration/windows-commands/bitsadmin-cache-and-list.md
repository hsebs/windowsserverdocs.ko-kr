# <a name="bitsadmin-cache-and-list"></a>bitsadmin 캐시 및 목록

모든 캐시 항목을 나열합니다.

## <a name="syntax"></a>구문

```
bitsadmin /cache /list [/verbose]
```

## <a name="examples"></a><a name=BKMK_examples></a>예와

다음 예제에서는 상세 형식에서 모든 캐시 항목을 나열합니다.

```
C:\> bitsadmin /cache /list /verbose
```
