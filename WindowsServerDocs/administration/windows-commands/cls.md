# <a name="cls"></a>cls

명령 프롬프트 창을 지웁니다.

## <a name="syntax"></a>구문

```
cls
```

### <a name="parameters"></a>매개 변수

| 매개 변수 | 설명 |
| --------- | ----------- |
| /? | 명령 프롬프트에 도움말을 표시합니다. |

## <a name="examples"></a>예

명령 프롬프트 창에 표시 되는 모든 정보를 지우고 빈 창으로 돌아가려면 다음을 입력 합니다.

```
cls
```
