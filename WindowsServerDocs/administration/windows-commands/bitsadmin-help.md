# <a name="bitsadmin-help"></a>bitsadmin help

Bitsadmin 명령줄 매개 변수 및 옵션에 대 한 도움말 관련 정보를 표시 합니다.

## <a name="syntax"></a>구문

```
bitsadmin /help | /?
```

## <a name="examples"></a>예

명령줄 도움말을 검색 합니다.

```
bitsadmin /help
```
