# <a name="bitsadmin-help"></a>bitsadmin help

Bitsadmin 명령줄 매개 변수 및 옵션에 대 한 도움말 관련 정보를 표시 합니다.

## <a name="syntax"></a>구문

```
bitsadmin /help | /?
```

## <a name="examples"></a><a name=BKMK_examples></a>예와

다음 예제에서는 명령줄 도움말을 검색합니다.

```
C:\>bitsadmin /help
```
