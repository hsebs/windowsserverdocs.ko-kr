# <a name="begin-restore"></a>복원 시작

> 적용 대상: Windows Server 2012 R2, Windows Server 2012

복원 세션 및 문제는 **PreRestore** 관련 된 작성기에는 이벤트입니다.

## <a name="syntax"></a>구문

```
begin restore
```
