# <a name="online"></a>온라인



오프 라인 디스크나 볼륨을 온라인 상태로 됩니다.

## <a name="syntax"></a>구문

```
Offline disk
Offline volume
```

### <a name="parameters"></a>매개 변수

|매개 변수|설명|
|---------|-----------|
|[오프 라인 디스크](offline-disk.md)|포커스가 있는 오프 라인 디스크를 온라인 상태로 됩니다.|
|[오프라인 볼륨](offline-volume.md)|포커스가 있는 오프 라인 볼륨을 온라인 상태로 됩니다.|
