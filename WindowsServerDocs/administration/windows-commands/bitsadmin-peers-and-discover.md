# <a name="bitsadmin-peers-and-discover"></a>bitsadmin 피어 및 검색

피어를 다시 검색합니다.

## <a name="syntax"></a>구문

```
bitsadmin /peers /discover
```

## <a name="examples"></a><a name=BKMK_examples></a>예와

다음 예제에서는 피어를 검색합니다.

```
C:\>bitsadmin /peers /discover
```
