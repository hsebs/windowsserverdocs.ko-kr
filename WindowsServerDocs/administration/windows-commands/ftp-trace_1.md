# <a name="ftp-trace_1"></a>ftp: trace_1



패킷 추적을 설정/해제 합니다.

## <a name="syntax"></a>구문

```
trace
```

#### <a name="parameters"></a>매개 변수

none

## <a name="remarks"></a>설명

**추적** 일련의 내부 FTP 함수 호출을 실행 하는 경우 표시는 **ftp** 명령입니다.

## <a name="examples"></a>예

추적 및 해제를 전환 합니다.
```
trace
```
