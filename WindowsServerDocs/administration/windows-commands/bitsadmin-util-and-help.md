# <a name="bitsadmin-util-and-help"></a>bitsadmin util and help

**/Sva** 스위치에 대 한 명령줄 사용법을 표시 합니다.

## <a name="syntax"></a>구문

```
bitsadmin /util /help
```

## <a name="examples"></a>예

**/Sva** 스위치에 대 한 명령줄 도움말을 표시 하려면 다음을 수행 합니다.

```
bitsadmin /util /help
```
