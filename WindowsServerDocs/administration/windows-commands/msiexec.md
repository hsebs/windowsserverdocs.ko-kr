# <a name="msiexec"></a>msiexec



## <a name="msiexec"></a>Msiexec

설치, 수정 및 명령줄에서 Windows Installer에 대 한 작업을 수행 하는 방법을 제공 합니다.

구문 및 예제는이 명령을 사용 하는 방법에 대 한 참조 [Msiexec](https://go.microsoft.com/fwlink/?LinkId=94329)합니다.
