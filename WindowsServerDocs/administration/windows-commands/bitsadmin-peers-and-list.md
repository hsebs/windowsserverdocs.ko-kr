# <a name="bitsadmin-peers-and-list"></a>bitsadmin peers and list

모든 피어를 나열 합니다.

## <a name="syntax"></a>구문

```
bitsadmin /peers /list
```

## <a name="examples"></a>예

모든 피어를 나열 하려면:

```
bitsadmin /peers /list
```
