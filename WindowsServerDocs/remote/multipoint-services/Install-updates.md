# <a name="install-updates"></a>업데이트 설치
사용 가능한 경우 업데이트를 설치 하는 것이 좋습니다. 업데이트를 설치 하려면 인터넷에 연결 해야 합니다.  

1.  **시작** 화면에서 **제어판**을 엽니다.  
  
2.  제어판에서 **업데이트**를 입력 한 다음 **업데이트 확인**을 클릭 합니다.  
  
3.  Windows 업데이트 웹 사이트에서 컴퓨터에 필요한 업데이트를 모두 나열 하는 경우 업데이트를 설치 합니다.
