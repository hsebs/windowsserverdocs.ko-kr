# <a name="prerequisites-for-deploying-directaccess"></a>DirectAccess를 배포하기 위한 필수 조건

>적용 대상: Windows Server(반기 채널), Windows Server 2016

다음 표에서는 구성 마법사를 사용 하 여 DirectAccess를 배포 하는 데 필요한 필수 구성 요소를 나열 합니다.
